## Run Locally

Clone the project

```bash
  git clone https://github.com/ZinebSafira/Portfolio-2024-v2.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
